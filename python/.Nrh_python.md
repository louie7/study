    ### 7. decorators: a function use function as input arguments
           to add addition feature without modify original function

    #### function decorator
    ```python
        import time
        def decorator(func):
            def wrapper(*args, **kwargs):
                start_time = time.time()
                func()
                end_time = time.time()
                print(end_time - start_time)

            return wrapper

        @decorator
        def func():
            time.sleep(1)

        func()
    ```
